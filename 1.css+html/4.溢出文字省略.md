## 溢出文字省略
写一下备忘把 




### 单行文字溢出省略
```
{
    overflow: hidden;
    text-overflow:ellipsis;
    white-space: nowrap;
}
```

### 多行文字溢出省略
WebKit的CSS扩展属性，该方法适用于WebKit浏览器及移动端
```
{
    display: -webkit-box;
    -webkit-box-orient: vertical;   //orient朝向
    -webkit-line-clamp: 3;
    overflow: hidden;
}
```
